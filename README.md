# 免費打造結合即時通知&店家推廣的問卷程式

### 也許您曾經有這樣的經驗：  
顧客填寫完問卷後留下負評但店家未即時處置  
顧客用餐體驗極佳但您卻沒讓他更近一步地幫店裡宣傳  

### 專案目標：
1. 客戶留下負面評價時會即時向店長(老闆)的line發出通知，讓店長(老闆)可以馬上了解狀況並處理
2. 客戶留下好的用餐評價會引導他幫店家粉絲團按讚留下好評

### 本文章使用到的技術：
1. Google Form表單
2. Google App Script
3. LINE Notify
